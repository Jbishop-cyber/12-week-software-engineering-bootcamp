## ⭐ Mini Project

### Student Score Statistics

A console application that:

- Accepts the number of students.
- Validates the number of students.
- Accepts each student's name and score.
- Validates that scores are between 0 and 100.
- Calculates:
  - Total number of students
  - Highest score
  - Lowest score
  - Average score
- Displays a formatted class report.

## 📌 Lessons Learned

- Use `while` loops when validating user input.
- Use `for` loops when the number of iterations is known.
- Proper indentation is critical in Python.
- Validate user input before processing it.
- Python provides powerful built-in functions that simplify common tasks.
