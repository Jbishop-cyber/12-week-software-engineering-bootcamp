# Week 1 - Day 2: User Input and Type Conversion

## 📚 Topics Covered

* `input()` function
* Accepting user input
* Data types returned by `input()`
* Type conversion (casting)

  * `int()`
  * `float()`
  * `str()`
* Formatted output using f-strings
* Basic arithmetic with user input

## 🎯 Skills Practised

## 💡 Key Takeaways

* `input()` always returns a **string**.
* Use `int()` or `float()` when numerical calculations are required.
* F-strings provide a clean and readable way to display variables.
* Meaningful variable names make code easier to understand and maintain.
