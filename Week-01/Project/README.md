# Week 1 Final Project – Student Grade Report System

## 📖 Overview

The **Student Grade Report System** is a command-line Python application developed as the final project for **Week 1** of my Software Engineering Bootcamp.

The program allows a user to enter information for multiple students, automatically calculates each student's grade and remark, displays an individual report for every student, and finally generates a class summary showing overall statistics.

This project combines all the Python concepts learned during Week 1.

---

## 🚀 Features

* Accepts the total number of students.
* Validates the number of students entered.
* Accepts each student's:

  * First Name
  * Surname
  * Score
* Validates that scores are between **0** and **100**.
* Calculates letter grades automatically.
* Displays a formatted report for every student.
* Calculates and displays:

  * Total number of students
  * Highest score
  * Lowest score
  * Average score
