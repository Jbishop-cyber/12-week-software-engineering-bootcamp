### Overview

Today's lesson focused on searching for information stored inside lists.

Learning how to search for specific items, determine whether an item exists, find its position, count occurrences, and validate user input. I also learned how to organize a search program using functions and loops.

---

## Topics Covered

- Searching through a list
- Searching with indexes
- Flag variables
- break statement
- Counting occurrences
- Finding the largest value
- Input validation
- Using strip()
- Building reusable functions

## Exercises Completed

- Search Student
- Find Student Index
- Count Occurrences
- Find Largest Number
- Student Exists

## Mini Project

### Student Search System

Features:

- Enter the number of students
- Store student names
- Display all students
- Display the first and last student
- Search for students by name
- Display the student's index
- Continue searching until the user exits

- Searching algorithms

## Lessons Learned

- A search should stop once the required item is found.
- Counting requires checking every element.
- Flag variables help track program state.
- `break` improves efficiency.
- `strip()` helps clean user input before validation.
- Breaking a problem into smaller functions makes programs easier to understand and maintain.
