## 💡 Key Takeaways

* Tuples are immutable and useful for fixed collections of data.
* Dictionaries store information using keys instead of indexes.
* `get()` is a safer way to retrieve values because it avoids `KeyError`.
* Dictionaries are often a better choice than lists for storing related information, such as student records or inventory.
* Breaking a program into small functions makes it easier to build, test, and maintain.
