# 12-Week Software Engineering Bootcamp

Welcome to my **12-Week Software Engineering Bootcamp** repository.

This repository documents my journey as I learn the fundamentals of software engineering through daily lessons, coding exercises, mini projects, and weekly projects. It serves as both a learning journal and a portfolio of my progress.

---

## 🎯 Objectives

The goals of this bootcamp are to:

* Build a strong programming foundation using Python.
* Develop problem-solving and algorithmic thinking.
* Learn software engineering best practices.
* Improve code organization using functions and modular design.
* Build progressively larger projects by applying newly learned concepts.
* Maintain consistent coding habits through daily practice.
